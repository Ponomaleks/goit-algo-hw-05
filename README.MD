## 2. Висновки

На основі отриманих даних було проведено порівняння ефективності трьох алгоритмів: Боєра-Мура, Кнута-Морріса-Пратта (KMP) та Рабіна-Карпа.

**Аналіз результатів:**

1.  **Боєр-Мур — найефективніший алгоритм.**
    Він продемонстрував найкращу швидкодію у всіх тестових сценаріях. Особливо показовим є результат пошуку неіснуючого підрядка, де Боєр-Мур випередив найближчого конкурента (KMP) у 5-7 разів, а Рабіна-Карпа — більш ніж у 10 разів. Це досягається завдяки евристиці "поганого символу", яка дозволяє алгоритму робити значні стрибки по тексту, пропускаючи завідомо невідповідні ділянки.

2.  **Кнут-Морріс-Пратт (KMP) — стабільний середній результат.**
    Алгоритм KMP показав кращі результати, ніж Рабін-Карп, але стабільно поступається Боєру-Муру. Хоча він ефективніший за наївний пошук (не повертається назад по тексту), відсутність механізму пропуску груп символів робить його повільнішим за лідера на великих текстах.

3.  **Рабін-Карп — найменш ефективний для даної задачі.**
    Алгоритм показав найгірший час виконання у всіх тестах. Витрати ресурсів на постійний перерахунок хеш-функції для кожного вікна перевищують вигоду від зменшення кількості посимвольних порівнянь при пошуку *одного* патерну.

4.  **Специфіка даних ("Стаття 2").**
    У другому тексті час знаходження існуючого підрядка є аномально малим і майже однаковим для всіх алгоритмів. Це свідчить про те, що шуканий фрагмент знаходився на самому початку файлу, що нівелювало різницю в асимптотичній складності алгоритмів.

**Підсумок:**
Для задач пошуку підрядка в текстових масивах даних **алгоритм Боєра-Мура є найбільш оптимальним вибором** з точки зору швидкодії. Використання алгоритму Рабіна-Карпа для пошуку одиничних підрядків є недоцільним.
